# Landind-page-projeto

# 📚 Integra Leitura

Uma landing page simples e acessível criada para apresentar o projeto **Integra Leitura**, com foco em inclusão social por meio da educação e tecnologia. Projeto desenvolvido como objeto do TCC do curso.

---

## 🧠 Sobre o Projeto

O **Integra Leitura** é um projeto baseado nas **ODS 4, 10 e 16 da ONU**, com o objetivo de promover:

* 📖 Acesso à educação de qualidade
* 🤝 Inclusão social de pessoas com deficiência
* ⚖️ Redução das desigualdades

A proposta é oferecer uma plataforma interativa com conteúdos adaptados para diferentes faixas etárias, desde estudantes até adultos que não concluíram seus estudos.

## 🎬 Efeito de Animação

A página utiliza JavaScript para aplicar um efeito de **fade-in ao rolar a página**, tornando a experiência mais dinâmica.

Funcionamento:

* Os elementos começam invisíveis (`opacity: 0`)
* Ao entrarem na tela, recebem a classe `.show`
* A transição é aplicada com `transition` no CSS
